## 贝塞尔曲线
> 四个点计算出一条曲线
```java
     private float bezier(float t, float p0, float p1, float p2, float p3) {
            return (1 - t) * (1 - t) * (1 - t) * p0
                    + 3 * (1 - t) * (1 - t) * (t) * p1
                    + 3 * t * t * (1 - t) * p2
                    + t * t * t * p3;
        }
```
